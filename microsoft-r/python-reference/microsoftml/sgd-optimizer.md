# *microsoftml.sgd_optimizer*: Stochastic gradient descent


**Applies to: SQL Server 2017 RC2**


## Usage

    momentum: numbers.Real = None, nag: bool = None,
    weight_decay: numbers.Real = None,

    l_rate_red_freq: numbers.Real = None,
