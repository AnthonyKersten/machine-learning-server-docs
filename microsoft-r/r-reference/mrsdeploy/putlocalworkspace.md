 #putLocalWorkspace: Copy the workspace of the local R session.

 Applies to version 1.1.0 of package mrsdeploy.
 
 ##Description
 
Copy all objects from the local R session and load them into the remote R session.
 
 
 ##Usage

```   
  putLocalWorkspace()
 
```
